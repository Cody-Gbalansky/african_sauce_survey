# African Sauce Survey

A cleaned-up React + TypeScript survey page based on the original single-file component.

## File structure

```txt
african-sauce-survey/
  index.html
  package.json
  tsconfig.json
  README.md
  src/
    App.tsx
    main.tsx
    components/
      survey/
        OptionButton.tsx
        ProductCard.tsx
        Progress.tsx
        Survey.tsx
        ThankYou.tsx
    constants/
      colors.ts
    data/
      products.ts
      questions.ts
    types/
      survey.ts
```

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
